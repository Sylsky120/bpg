變數/運算子

變數(Variable)簡介

變數の定義：一個有名稱的記憶體儲存空間，其中可包含資料型態、記憶體位址、儲存值。 Tips: 變數名稱盡量取英文名稱！


變數の宣告：


 #include<iostream>
 using namespace std;
 int main(){
   int number1;
   int numder2 = 10;
 //資料型態 變數名稱 (=儲存值);
   float number3 = 10.5;
   return 0;
 }

常用變數類型

整數型態：

  • int
  • long long
  • unsidned long long

int的可儲存範圍最小，相對的其占用之記憶體空間也最小。
反之unsigned long long的可儲存範圍最大，相對的其占用之記憶體空間也最大。



浮點數(小數)型態：

  • float
  • double
  • double double

float型態從小數點後開始計算，可儲存的位數最少，精確度最低。
反之，double double型態的精確度最高。



布林值型態：bool

常用於是非判斷。



字元/字串型態：

  • char 字元
  • string 字串

字串是由一個以上的字元所組成。


變數の範圍和用法：


#include<iostream>
using namespace std;
int main(){
    int int1 = 10;  //範圍: -2147483648 ~ 2147483647
    int int2 = 10000000000;
    long long ll = 100;  //範圍: -9223372036854775808 ~ 9223372036854775807
    unsigned long long ull = 1000;  //範圍: 0 ~ 18446744073709551615
    cout << "\n int1 = " << int1 << '\n';
    cout << "\n int2 = " << int2 << '\n';
    cout << "\n ll = " << ll << '\n';
    cout << "\n ull = " << ull << '\n';
   
    float float1 = 1.23; //精度: 3.4E +/- 38 (7 位數)

常見運算子

算術運算子：

運用於一般的數學計算。

  • + → 加
  • - → 減
  • * → 乘
  • / → 除(取商數，無條件捨去)
  • %(mod) → 取餘數


指派運算子：

原本的值根據運算符號和另一個值做出改變後，再指派給原先的變數。

  • A+=B → A的值變成(A+B)的值
  • A-=B → A的值變成(A-B)的值
  • A*=B → A的值變成(A*B)的值
  • A/=B → A的值變成(A除以B之商數)的值
  • A%=B → A的值變成(A除以B之餘數)的值

也就是說，「A+=B」 等同於 「A=A+B」 。



關係運算子：

對兩個值做出比較的行為，再回傳一個 True or False 的結果。

  • == → 等於
  • != → 不等於
  • >= → 大於等於
  • > → 大於
  • <= → 小於等於
  • < → 小於

判斷式

如果-(否則如果)-否則 "if-(else if)-else"


"if-else" 用於程式需要依照不同的條件，執行不同結果時。

基本語法：


#include<iostream>
using namespace std;
int main(){
    if(條件1成立){
        執行程式1;
    }
    else if(條件2成立){     //可省略
        執行程式2;
    }
    else{                 //可省略
        執行程式3;
    }
    return 0;
}

注意縮排，執行程式每行都需縮排。
注意格式，"if", "if else", "else"各為一個區塊(每個區塊都擁有屬於自己的大括號，不存在誰包住誰)。


範例程式碼：


#include<iostream>
using namespace std;
int main(){   
    int Α, Β;
    cin >> Α >> Β;
    if(Α==Β){              //如果A等於B
        cout << ″the same\n″;     //輸出"the same"
    }
    else if(Α > Β){        //如果A大於B
        cout << Α << ′\n′;       //輸出A的值
    }
    else if(Α < Β){        //如果A小於B
        cout << Β << ′\n′;       //輸出B的值
    }
    else{                  //否則
        cout << ″nothing\n″;     //輸出"nothing"
    }
    return 0;
}

#include<iostream>
#include<math.h> ⁄⁄可以用sqrt()的標頭檔
using namespace std;
int main(){
    int a, b, c;
    cin >> a >> b >> c;
    ⁄⁄ x=(−b±√b*b−4ac)/2a
    int tmp = b*b-4*a*c;
    if(tmp > 0){  
        int x1=(-b+sqrt(tmp))/(2*a); ⁄⁄sqrt() 開根號
        int x2=(-b-sqrt(tmp))/(2*a);
        if(x1 < x2){  ⁄⁄轉換x1、x2
            int k;
            k = x1;
            x1 = x2;
            x2 = k;
        }
        cout << "Two different roots x1=" << x1 << " , x2=" << x2 << '\n';
    }
    else if(tmp == 0){
        int x=(-b)/(2*a);
        cout << "Two same roots x=" << x << '\n';
    }
    else{
        cout << "No real root\n";
    }
    return 0;
}

"switch-case"

"switch-case" 用於一個變數(數字 or 字元)等於不同值，執行不同的結果。

switch 的括號內放條件式，是不被允許的!  EX.switch(A>10)

基本語法：


#include<iostream>
using namespace std;
int main(){
    switch(變數名稱 or 運算式){
        case 符合的數字或字元:
            執行程式1;
            break;
        case 符合的數字或字元:
            執行程式2;
            break;
        default:  //默認
            break;
    }
    return 0;
}

範例程式碼：


#include<iostream>
using namespace std;
int main(){
    cout << "Please enter Your grade...\n";
    char grade;
    cin >> grade;
    switch(grade){
        case 'A':
            cout << "Perfect!\n";
            break;
        case 'B':
        case 'C':
            cout << "Very Good!\n";
            break;
        case 'D':
            cout << "You Pass!\n";
            break;
        case 'F':
            cout << "You Fail!\n";
            break;
        default:
            cout << "Please enter your grade by A, B, C, D, F...\n";
    }
   return 0;
 }

迴圈

當(...)時(...) "while"


"while" 常用於某段程式需要多次執行的時候。

需要注意的是，當括號內的條件成立時，while迴圈會一直重複執行。
也就是說，如果執行程式沒有設置停止的條件，整個迴圈會無限次重複執行。

基本語法：


#include<iostream>
using namespace std;
int main(){
    while(條件成立){
        執行程式;
    }
    return 0;
}

範例程式碼：


#include<iostream>
using namespace std;
int main(){
    ⁄⁄輸出 n~1
    int n;
    cin >> n;
    while(n >= 1){
        cout << n << '\n';
        n–=1;
    }
    return 0;
}

"for" 迴圈


基本語法：


#include<iostream>
using namespace std;
int main(){
    for(前情提要; 執行條件; 收尾工作){
        執行程式;
    }
    return 0;
}

前面的程式 加上 "前情提要" ，符合 "執行條件" ，則 "執行程式" ，"收尾工作"，準備開始下一回合。
直到 "執行條件" 不通過 or  "執行程式" 被中斷時，結束整個迴圈。

注意!  "前情提要"只有在第一輪的迴圈會執行。


範例程式碼：


#include<iostream>
using namespace std;
int main(){
    int n;
    cin >> n;
    for(int i=n; i>=1; --i){
        cout << i << '\n';
    }
    return 0;
}

最大公因數(GCD)
原始暴力法(解這題會TLE) 
#include<iostream>
#include<algorithm>
using namespace std;
int main(){
    int t;
    cin >> t;
    while(t--){
        int a, b;
        cin >> a >> b;
        if(a < b){  ⁄⁄確保a大於b
            swap(a, b);
        }
        int gcd=0;
        for(int i=1; i<=b; ++i){
            if(a%i==0 && b%i==0){
                gcd = i;
            }
        }
        cout << gcd << '\n';
    }
    return 0;
}
輾轉相除法 
#include<iostream>
using namespace std;
int main(){
    int t;
    cin >> t;
    while(t--){
        int a, b;
        cin >> a >> b;
        while(a!=0 && b!=0){  ⁄⁄輾轉相除法
            if(a >= b){
                a %= b;
            }
            else{
                b %= a;
            }
        }
        if(a >= b){
            cout << a << '\n';
        }
        else{
            cout << b << '\n';
        }
    }
